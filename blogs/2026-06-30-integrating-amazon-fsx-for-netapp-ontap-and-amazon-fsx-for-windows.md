This post explains how to connect FSx for NetApp ONTAP and FSx for Windows File Server to Microsoft Entra ID using Entra Domain Services, letting organizations retire Active Directory in favor of Entra ID while keeping SMB authentication working. It covers configuring reverse DNS records, creating file systems, joining storage virtual machines to managed domains, and validating user authentication across hybrid AWS-Azure environments.
